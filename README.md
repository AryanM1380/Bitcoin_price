# Bitcoin Price Checker

## Overview

This Python script leverages the CoinGecko API to retrieve and display the current Bitcoin prices in various currencies. It supports USD, Euro (EUR), Iranian Rial (IRR), and Canadian Dollar (CAD). The script is a simple command-line tool providing real-time cryptocurrency prices.

## Requirements

- Python 3.x
- `requests` library

Install the required library using the following command:

```bash
pip install requests
```

## Usage

1. Clone the repository or download the `bitcoin_price.py` file.

2. Open a terminal or command prompt.

3. Navigate to the directory containing the `bitcoin_price.py` file.

4. Run the script using the following command:

```bash
python bitcoin_price.py
```

5. The script will fetch and display the current Bitcoin prices in USD, Euro, Iranian Rial, and Canadian Dollar.
